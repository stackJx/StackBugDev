## 重点分布

1. **理论基础**：CAP、BASE、共识算法（Paxos / Raft）、向量时钟
2. **消息队列**：Kafka / RocketMQ / RabbitMQ 的模型差异；如何保证不丢/不重/有序
3. **分布式缓存**：一致性哈希、热 key、缓存与 DB 的一致性策略（旁路、双写、订阅 binlog）
4. **分布式事务**：2PC、TCC、Saga、消息事务

## 复习路径

从"为什么需要分布式"开始 → 网络分区下的折中（CAP）→ 一致性模型 → 具体中间件的实现。**所有方案都是 trade-off**，能讲清楚利弊比记结论更重要。

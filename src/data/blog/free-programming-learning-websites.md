# 计算机科学与编程学习资源导航

## 📚 数据结构与算法

- Hello Algohttps://www.hello-algo.com/动画图解 + 一键运行的数据结构与算法教程，适合零基础入门与进阶学习
- 小林编程 https://xiaolincoding.com/ 图解计算机网络、操作系统、计算机组成、数据库，用可视化方式攻克八股文考点

## ☕ Java 全栈开发

- JavaGuidehttps://javaguide.cn/Java 学习与面试指南，覆盖核心知识体系与高频面试题
- 虫洞栈https://bugstack.cn/实战型教程合集：涵盖 Java 基础、Spring 源码、设计模式、JVM 实现、Netty 开发等
- 深度 Java 面经 https://www.cnblogs.com/lidong422339/p/17392237.html Java 高阶面试经验与底层原理分析
